## 🌟 为什么跨境卖家需要它？

在亚马逊、Shopee、eBay 等平台，**纯[白底图片](https://pictech.cc)**是合规上架的刚需。手动抠图耗时耗力，批量处理又是痛点。

本 Skill 是 **[妙言小智 (PicTech.cc)](https://pictech.cc)** 为 OpenClaw/WorkBuddy 深度定制的生产级工具。它不仅能精准去除背景，还能根据指令一键填充任意颜色，是电商图片上架、详情页设计的效率神器。

### 核心亮点：
*   **电商级边缘处理**：对产品边缘、发丝、半透明材质有极佳的识别能力，边缘平滑不生硬。
*   **一键白底图**：支持参数化背景填充。只需一句“帮我换成白底”，即可输出符合电商平台规范的 255 纯白背景图。
*   **Agent 深度集成**：
    *   **图片即时渲染**：无需打开文件夹，[抠图](https://pictech.cc)结果直接以预览图形式呈现在对话框。
    *   **VK 持久化记忆**：首创“一次输入，永久记住”，后续调用无需再次寻找密钥。
*   **本地高效存档**：自动按“日期-批次”对处理结果进行物理归档，支持 SQLite 本地缓存，重复素材不扣额度。

## 📋 功能参数

| 参数名 | 必填 | 说明 |
| :--- | :--- | :--- |
| `input` | 是 | 支持单个路径、URL 或包含图片的文件夹路径。 |
| `backgroundColor` | 否 | 背景颜色。默认为透明。支持 `white`, `red` 或 `#FFFFFF`。 |
| `vk` | 首次 | 妙言小智 API Key。请前往 [PicTech.cc](https://www.pictech.cc/) 申请。 |
| `saveDir` | 否 | 自定义保存目录。默认为用户目录下的 `myxz-result/bgremove`。 |

## 🔑 凭据记忆逻辑

本 Skill 采用了 **“静默授权”** 协议：
*   用户第一次使用时提供 VK，脚本会自动将其加密保存在本地 `.vk_config_rmbg` 文件中。
*   后续调用时，AI Agent 会自动读取本地配置，用户**无需再次输入 VK**。
*   **若无 VK**：AI 会自动引导用户前往 **[妙言小智官网 (PicTech.cc)](https://www.pictech.cc/)** 申请。
